# Glossário e Referências

## 1. Termos Técnicos
* **SAA**: Sistema de Apoio à Alocação Ambulatorial.
* **MVP**: Minimum Viable Product, com foco em Visualização e Apoio à Decisão neste contexto.
* **AGHU**: Aplicação de Gestão para Hospitais Universitários.
* **CSV**: Formato de arquivo que armazena dados em tabela.
* **Frontend**: Parte visível de uma aplicação.
* **Backend**: Parte não visível de uma aplicação
* **Vue**: Framework JavaScript para criação de interfaces de usuário
* **API**: Application Programming Interface, conjunto de regras e protocolos
* **FastAPI**: FrameWork python para criação de APIs
* **JWT**: JSON Web Token
